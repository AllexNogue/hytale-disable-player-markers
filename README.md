# hytale-disable-player-markers
A lightweight and focused Hytale server plugin that disables player map markers, preventing players from tracking each other through the map.

This plugin is designed for PvP, anarchy and hardcore servers, where exploration, positioning and awareness should come from gameplay â€” not from the map UI.

Only player markers are affected.
All other map elements and markers remain untouched.
